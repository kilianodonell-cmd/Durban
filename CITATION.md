## Reproducibility Information

To reproduce results from published work using this code:

1. Reference the exact version used: `git commit [hash]`
2. Document all input data sources and dates
3. Provide analysis parameters from your `config.json`
4. Include your edited factor definitions (if any)
5. Report the Python and library versions used

If you publish research using this analysis, please:
